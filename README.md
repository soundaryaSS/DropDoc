# Document Portal (Local)

This is a minimal Node/Express backend with a small React frontend (served via CDN) for user registration/login and viewing/downloading documents stored locally.

Quick start:

1. Open a terminal in `server` and install dependencies:

```bash
cd server
npm install
```

2. Start the server:

```bash
npm start
```

3. Open http://localhost:4000 in your browser.
